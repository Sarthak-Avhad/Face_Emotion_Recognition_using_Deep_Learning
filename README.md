# 😊 Face Emotion Recognition using Deep Learning

## 📌 Project Overview

This project focuses on building a **Deep Learning model** to recognize **human emotions from facial expressions**.  
Using image data, the model classifies facial expressions into different emotion categories such as happy, sad, angry, surprised, etc.

The project demonstrates the power of **Convolutional Neural Networks (CNNs)** in solving computer vision problems.

---

## 🎯 Objectives

- Build an emotion detection model using deep learning  
- Understand image preprocessing techniques  
- Train a CNN model for classification  
- Evaluate model performance on facial emotion data  

---


## 🔧 Model Used

- Convolutional Neural Network (CNN)  
- Activation Functions: ReLU, Softmax  
- Loss Function: Categorical Crossentropy  
- Optimizer: Adam  

---

## 🛠️ Technologies Used

- Python  
- Jupyter Notebook  
- TensorFlow / Keras  
- OpenCV  
- NumPy  
- matplotlib  

---

## 🔹 Project Workflow

1. Load and explore image dataset  
2. Perform image preprocessing:
   - Resizing  
   - Normalization  
   - Data augmentation (if applied)  
3. Build CNN model architecture  
4. Train the model on training dataset  
5. Evaluate model performance  
6. Predict emotions from test images
